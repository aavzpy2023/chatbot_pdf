**[FUNDAMENTOS DEL SISTEMA]**

*   DESCRIPCIÓN: Introducción al Sistema Económico Integrado VERSAT-Sarasola y su enfoque en el procesamiento de documentos primarios para el registro de hechos económicos. El subsistema de "Complementos" añade valor a la funcionalidad y adaptabilidad del software, permitiendo a los usuarios obtener información para uso interno y para suministrar a diferentes niveles.
*   PREGUNTAS_CLAVE:

    *   ¿Cuál es la filosofía principal del sistema VERSAT-Sarasola?
    *   ¿Qué valor añadido proporciona el subsistema de "Complementos"?
    *   ¿Para qué tipos de usuarios está diseñado este software?
    *   ¿Qué tipos de informes se pueden generar con este sistema?
*   RESPUESTAS:

    *   El sistema se basa en el procesamiento de documentos primarios para registrar hechos económicos.
    *   El subsistema de "Complementos" mejora la funcionalidad y adaptabilidad del software. Permite a los usuarios obtener la información necesaria para uso interno y externo.
    *   El software está diseñado para distintos usuarios con diferentes requerimientos en el procesamiento y obtención de información.
    *   Se pueden generar informes tanto para uso interno de la entidad como para ser suministrados a diferentes niveles que requieran la información.

*   RESPUESTAS:
